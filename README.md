# [Blissy](https://github.com/blissful-group/blissy)

The simple auth library.

## Repository

Monorepository for the `@blissy` authentication packages.

This repository contains the packages and shared code used to build auth for the `@blissy` ecosystem.

## Getting Started

Install `proto` by following the official documentation:

https://moonrepo.dev/docs/proto/install

Then install the repository's managed tools:

```sh
proto install
```
